Monitoring

Welkom op de overzichtspagina van ons eindwerk. Hier kan je makkelijk de lijsten oproepen van de gebruikers per IP, per SSID of per AP. Kies in het menu vanboven voor een van deze voor een voorbeeld.


Hier kan je zoeken. Zoeken is mogelijk op IP/MAC/Username. Je kan bijvoorbeeld een studentennummer ingeven om die student te zoeken, @UCLL voor al de studenten te zien die hiermee zijn aangemeld, een deel van een IP voor al de IPs te vinden in dat subnet.

Je kan ook simpele regexen gebruiken. Hieronder enkele voorbeelden:
Wat Regex
Zoeken r-nummer: r[0-9]+@
Zoeken u-nummer u[0-9]+@
Zoeken domein .[0-9]+@ucll
Zoeken IP [0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}
